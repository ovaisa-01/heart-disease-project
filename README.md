# Heart Disease Prediction (Beginner Project)

## Overview
Simple machine learning project to predict heart disease using logistic regression.

## Files
- `heart_disease.py` : main script
- `heart.csv` : dataset (place this file in the same folder)
- `requirements.txt` : Python packages
- `.gitignore` : ignore rules
